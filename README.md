# Для запуска проекта
1. Укажите доступы к базе дыннвх .env

2. Запустите миграции
``` php artisan migrate ```

3. Запустите сиды
``` php artisan db:seed ```

4. Cоздайте пользователя и скопируйте Bearer токен 
``` php artisan user:create ```

5. Запустить тесты
``` php artisan test ```
